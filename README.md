# Yooz Homebrew Tap

Homebrew formulae for [Yooz](https://github.com/yooz-labs) packages.

## Install

```bash
brew tap yooz-labs/tap
brew install remi
```

## Available formulae

| Formula | Description |
|---------|-------------|
| `remi` | Remote monitor for Claude Code CLI sessions |

## Update

```bash
brew update
brew upgrade remi
```
